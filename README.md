# Kaomoji Protocol 🎭

Децентрализованная биржа для торговли эмодзи-токенами в Telegram.

## Особенности

- 📊 ASCII-графики в реальном времени
- 💰 Торговля каомодзи-токенами ((^_^), (⌐■_■) и др.)
- 🔄 Интеграция с TON DEX (STON.fi, Dedust)
- 🏦 Некастодиальные кошельки (@wallet)
- 📱 Простой интерфейс в Telegram

## Установка

1. Установите зависимости:
\\\ash
npm install
\\\

2. Настройте окружение:
\\\ash
cp .env.example .env
# Отредактируйте .env файл
\\\

3. Запустите бота:
\\\ash
npm run dev
\\\

## Команды бота

- \/start\ - Главное меню
- \/trade\ - Торговая площадка
- \/portfolio\ - Портфель
- \/admin\ - Админ-панель

## Структура проекта

\\\
src/

## Технологии

- Telegraf - Telegram Bot API
- TON Blockchain - смарт-контракты
- TypeScript - типобезопасность
- Node.js - серверная платформа
